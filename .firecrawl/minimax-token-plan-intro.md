## [​](https://platform.minimax.io/docs/token-plan/intro\#welcome-to-the-token-plan)  Welcome to the Token Plan!

MiniMax is one of the few AI labs that develops frontier models across the full spectrum of modalities: text, speech, video, music, and image. The [Token Plan](https://platform.minimax.io/subscribe/token-plan) extends upon our former Coding Plan by providing usage beyond language models, allowing for more creative agents and application to be built and used.

## [​](https://platform.minimax.io/docs/token-plan/intro\#core-advantages)  Core Advantages

1. **Full-Spectrum Multimodal Access** — Access models of multiple modalities under a single subscription, with M2.7 requests resetting every 5 hours and other models using daily quotas.
2. **Powered by the Latest M2.7 Model** — All plans include the latest MiniMax M2.7 model, with M2.7-highspeed availability based on resource load. High-Speed subscriptions offer dedicated M2.7-highspeed support for even faster inference.
3. **Extremely Cost-Effective** — A fixed-fee subscription grants you substantial usage quotas across multiple supported models, so you can build freely without unpredictable bills.

## [​](https://platform.minimax.io/docs/token-plan/intro\#usage-quota)  Usage Quota

The [Token Plan](https://platform.minimax.io/subscribe/token-plan) usage quota is calculated per model:

- **M2.7 / M2.7-highspeed**: Measured by requests, with a 5-hour rolling reset.
- **Other models** (speech, video, music, image): Measured by daily quotas, resetting daily.

## [​](https://platform.minimax.io/docs/token-plan/intro\#getting-started)  Getting Started

Only 2 steps to quickly activate your [**Token Plan**](https://platform.minimax.io/subscribe/token-plan) service**Step 1: Subscribe**Visit the [**Token Plan**](https://platform.minimax.io/subscribe/token-plan) Subscription page, choose the plan that best suits your needs and complete the subscription process.**Step 2: Get Token Plan API Key**Once your subscription is successful, navigate to the [**Account/Token Plan**](https://platform.minimax.io/user-center/payment/token-plan) page. Here you can view your active plan details and get your **Token Plan API Key**.

**Important Notes**

- This API Key is exclusive to the [Token Plan](https://platform.minimax.io/subscribe/token-plan) and is not interchangeable with the API Keys for pay-as-you-go text models.
- This API Key is only valid during the active period of your [Token Plan](https://platform.minimax.io/subscribe/token-plan) subscription.
- Please protect your API Key to prevent any loss of resources.

## [​](https://platform.minimax.io/docs/token-plan/intro\#after-reaching-the-usage-limit)  After Reaching the Usage Limit

Token Plan has two types of usage limits:**M2.7 — 5-Hour Rolling Window**When you reach the request limit within a 5-hour window, you have the following options:

1. **Switch to Pay-As-You-Go**:
If you wish to continue without rate limits, you can replace your Token Plan API Key with your [pay-as-you-go API key](https://platform.minimax.io/user-center/basic-information/interface-key). This will switch the tool to a pay-as-you-go model based on actual token usage, which will consume your API account balance.
2. **Wait for the Reset**:
The [Token Plan](https://platform.minimax.io/subscribe/token-plan)’s text usage limit is calculated on a rolling 5-hour window. You can simply pause usage and wait for your quota to be restored.

**Other Models — Daily Quotas**When your daily quota for a non-text model is exhausted, you have the following options:

1. **Switch to Pay-As-You-Go**: Use your pay-as-you-go API Key to continue calling the corresponding model.
2. **Wait for the Daily Reset**: Daily quotas for non-text models reset automatically each day.
